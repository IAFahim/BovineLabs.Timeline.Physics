# BovineLabs Timeline Physics

`com.bovinelabs.timeline.physics` extends BovineLabs Timeline with physics-oriented authoring components and runtime systems for DOTS projects.

## Included authoring types
- `PhysicsVelocityTrack`
- `PhysicsVelocityClip`
- `StatefulTriggerTrack`
- `PhysicsTriggerInstantiateClip`

## Included runtime types
- `PhysicsVelocityTrackSystem`
- `PhysicsVelocityMixer`
- `PhysicsTriggerInstantiateSystem`

## Sample location
- `Sample~/Physics`

For package setup, dependency versions, and usage notes, see the package `README.md`.
